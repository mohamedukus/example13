Media Queries

Media queries are a powerful tool in CSS that allow you to apply styles based on the viewport size and other characteristics of the device.

@media (max-width)

بالعربية: تُستخدم لتطبيق التنسيقات عندما يكون عرض الشاشة أصغر من قيمة معينة.
In English: Used to apply styles when the screen width is smaller than a specific value.

@media (min-width)

بالعربية: تُستخدم لتطبيق التنسيقات عندما يكون عرض الشاشة أكبر من أو يساوي قيمة معينة.
In English: Used to apply styles when the screen width is greater than or equal to a specific value.

@media (min-width) and (max-width)

بالعربية: تُستخدم معًا لتطبيق التنسيقات فقط عندما يكون عرض الشاشة بين قيمتين محددتين.
In English: Used together to apply styles only when the screen width falls between two specific values.
